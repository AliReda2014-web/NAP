# NAP - Notes App Pro #
## تطبيق ملاحظات بسيط وسريع ##
### ما هو NAP? ###
**NAP** هو موقع ملاحظات بسيط وسريع تم بناءه أولًا كمشروع برمجي ثم تطور لاحقًا لموقع ملاحظات.
### المطور ###
تم تطوير الموقع بواسطة المبرمج المصري **علي رضا**.
### التوكنولوجيا المستخدمة ###
- HTML
- CSS
- Vanilla JS
- Local Storage
